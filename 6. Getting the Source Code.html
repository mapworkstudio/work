Now that you have a code editor, and node installed on your system, you have the tools necessary to start coding in JavaScript!

The online github repository for this course will help you out. It has outlines of all of the lectures, answers to quizzes, and solution guides to coding challenges.

You can download a zip file of the course, clone it, or best of all, fork your own version of the repository if you have a github account. With your own forked version, you can make your own custom changes!

All right! With access to the accompanying repository, let’s dive into some JavaScript!
